# random

Tiny tools that generate random things. Deployed on [GitHub Pages](https://hoangkhoaa.github.io/random/).

## Tools

| Tool | Text | UI |
|------|------|----|
| 🎲 Dice | `/dice/` | `/dice/ui/` |
| 🪙 Coin | `/coin/` | `/coin/ui/` |
| 🔢 Number | `/number/` | `/number/ui/` |
| 🃏 Card | `/card/` | `/card/ui/` |
| 🎡 Wheel | `/wheel/` | `/wheel/ui/` |
| 🔑 Password | `/password/` | `/password/ui/` |

## Features

- **Text endpoints** — minimal HTML, auto-generate on load, shareable links
  - Number supports `?min=X&max=Y` query params
  - Wheel supports `?items=A,B,C` query params
- **UI endpoints** — animated, interactive, dark theme
